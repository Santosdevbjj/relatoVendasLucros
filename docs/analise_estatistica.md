# Análise Estatística — Relatório Vendas e Lucros

## 1. Introdução
A análise estatística permite extrair padrões e comportamentos a partir dos dados de vendas, avaliando correlações entre produtos, regiões e clientes.

## 2. Estatísticas Descritivas
- Média de vendas e lucros por região.
- Desvio padrão para medir variabilidade de receita.
- Percentis e quartis para análise de distribuição.

## 3. Correlações
Correlação positiva entre **quantidade vendida** e **lucro líquido**, com *r = 0.82*, indicando que produtos com maior volume geram lucros proporcionais.

## 4. Teste de Hipóteses
Aplicado teste t para comparar médias de lucro entre as regiões Sul e Sudeste.  
**Resultado:** p-valor < 0.05 → diferença estatisticamente significativa.

## 5. Conclusões
- Região Sudeste é a mais lucrativa.  
- Produtos eletrônicos apresentam margens superiores.  
- Clientes recorrentes aumentam o lucro médio em 18%.
